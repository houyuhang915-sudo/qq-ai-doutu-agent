# Mirror Lab UI Only

这是一份从主项目中单独提取出来的纯前端 UI 演示包。

## 运行方式

1. 在当前目录安装依赖
2. 启动开发环境

```bash
npm install
npm run dev
```

默认地址：

[`http://127.0.0.1:4174/`](http://127.0.0.1:4174/)

## 说明

- 这份提取版不依赖本地后端接口。
- 上传图片仅用于前端预览展示。
- 点击“生成专属回复方案”后，会使用本地 mock 数据刷新结果区。
- 主视觉资源与交互样式已完整保留，适合单独演示 UI、继续换皮，或接入你自己的后端。
